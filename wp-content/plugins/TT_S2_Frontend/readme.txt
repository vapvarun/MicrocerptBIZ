Requires at least: WordPress 3.0 
Tested up to: WordPress 3.3.1 
Stable tag: 0.1

== Description ==

Provides a front end form through the use of a shortcode to manage "Subscribe2" subscriptions without sending user to Wordpress Dshboard.


== Installation ==

Activate and add shortcode [S2FE] into the subscribe page. Have fun :)
