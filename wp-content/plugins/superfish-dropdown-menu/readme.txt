== Description ==

This is a navigation widget that can be used for pages & for categories. Dropdown is based on Superfish JQuery plugin by  [Joel Birch](http://users.tpg.com.au/j_birch/plugins/superfish). You can use as many as you like. For more information & more useful widgets [visit shailan.com](http://shailan.com).

== Installation ==

1. Upload the plugin through `Plugins > Add New > Upload` interface.
1. Activate the plugin once uploaded. 
1. Go to Widgets section and drag'n drop it anywhere you like.
1. Remember to save your widget once you change the content.
